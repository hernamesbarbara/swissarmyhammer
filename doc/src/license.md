## Dependencies

SwissArmyHammer uses various open-source dependencies, each with their own licenses:

### Core Dependencies

- **tokio** (MIT) - Async runtime
- **serde** (MIT/Apache-2.0) - Serialization framework
- **clap** (MIT/Apache-2.0) - Command line parsing
